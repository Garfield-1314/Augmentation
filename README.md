## 🔧 Features | 功能

- **Image Augmentation** - Multiple advanced augmentation techniques
- **Mask Processing** - Background compositing and foreground extraction
- **Dataset Handling** - Support for YOLO, VOC, and classification formats
- **Format Conversion** - XML ↔ VOC ↔ YOLO conversions

## ⚙️ Requirements | 要求

- Python 3.11.13
- OpenCV, Matplotlib, Albumentations, tqdm
